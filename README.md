# FinFemme

FinFemme is a gamified financial literacy platform designed to empower women through bite-sized, interactive lessons. Inspired by Duolingo, it makes learning personal finance fun and accessible, focusing on budgeting, investing, retirement, credit, and loans.

## Features

- Gamified Learning: Earn XP, maintain streaks, and use hearts to stay motivated while progressing through courses.
- Interactive Lessons: Multiple-choice and true/false questions with immediate feedback and explanations.
- Progress Tracking: Track your journey with levels, hearts, and completion status across multiple courses.
- User Authentication: Secure sign-up and sign-in using NextAuth.js with Google OAuth and credentials.
- Responsive Design: Optimized for mobile-first experience with Shadcn UI components.

## Tech Stack

- Frontend: Next.js 16 with App Router, React 19, Tailwind CSS, Shadcn UI
- Backend: NextAuth.js, Prisma ORM with SQLite database
- Animations: Framer Motion for smooth transitions
- Deployment: Vercel

## Getting Started

1. Clone the repository: git clone [https://github.com/aadils29/hackviolet26.git](https://github.com/aadils29/hackviolet26.git)
2. Install dependencies: npm install
3. Set up the database: npx prisma migrate dev
4. Run the development server: npm run dev
5. Open [local host](http://localhost:3000) to view the app
